# backdrop_agent_skills
A directory for Backdrop related AI agent skills

Started at Backdrop LIVE - April 24, 2026

Initially, this is a directory of individual skill collections that members of the 
Backdrop community are using to help guide their Backdrop work using AI agents. Our
goal is to share and learn from each other with a longer term goal of building 
an "official" set of Backdrop Skills available to members of the community who use
Backdrop in their work. 

Please, submit a PR adding a link to the Skills you wish to share.
